# Chris Studio v2.1.0

Chris Studio v2.1 focuses on a unified Codex-style conversation experience, product identity, macOS permissions and release reliability.

## Highlights

- Local Chris Studio identity response and support contact.
- One-click safe send, Cmd/Ctrl+Enter and visible request timing.
- In-conversation project, Git, checks, Skills and Computer Use commands.
- macOS Accessibility and Screen Recording permission probe.
- New obsidian/ivory/violet app icon and UI palette.
- Fixed nested desktop dependency installation in GitHub Actions.
- Replaced conflicting legacy CI/release workflows.

Community builds remain ad-hoc signed when Apple Developer credentials are not configured.
